# Mr. Simbull Bot

A Telegram chatbot based on Cleverbot API

### ü§ñ [Click here to have some fun with Mr. Simbull](http://t.me/simbull_bot) 

## Disclaimer

Simbull is oblivious to the existence of bad things. He might not be appropriate for children under age of 16. He always attempts to observe and imitate other humans. He can come out as rude or inappropriate to you as a result. So, speak at your own risk and with caution. He plays the part of a person on occasion. So, even if he asks, avoid giving him your personal information. Finally, he cannot mean whatever he says since he does not comprehend you (or your feelings).


## Dependencies

* [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot)
* [cleverwrap](https://pypi.org/project/cleverwrap/)

```
$ pip install python-telegram-bot
$ pip install cleverwrap
```

## Steps 

1. Download or clone the project
2. Create your telegram bot using [BotFather](https://t.me/BotFather)
3. Purchase an API key from [Cleverbot](https://www.cleverbot.com/api)
4. Open the project and rename `constants.py` with your Telegram Bot's and Cleverbot's API Keys.
5. Run `main.py`
6. Boom! Your chatbot is ready now.

## APIs Used

- Cleverwrap (to interact with the official CleverBot API)
- Gizoogle API (extra: Other APIs)
